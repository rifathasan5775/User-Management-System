All the URLs
------------------

1. Create Route (POST): http://localhost:5000/api/users/create-user

2. Read Route (GET): http://localhost:5000/api/users/read-user

3. Update Route (PUT): http://localhost:5000/api/users/update-user

4. Delete Route (DELETE): http://localhost:5000/api/users/delete-user

5. Get All Users (GET): http://localhost:5000/api/users/all-users

6. Get User By ID (GET): http://localhost:5000/api/users/user/(User ID)

7. User Login (POST): http://localhost:5000/api/users/login

8. User Logout (POST): http://localhost:5000/api/users/logout

9. Change Password (PUT): http://localhost:5000/api/users/change-password

10. Update Profile (PUT): http://localhost:5000/api/users/update-profile

11. Make Admin (PUT): http://localhost:5000/api/users/make-admin/(User ID)

12. Remove Admin (PUT): http://localhost:5000/api/users/remove-admin/(User ID)

13. Search Users (GET): http://localhost:5000/api/users/search

14. Filter Users (GET): http://localhost:5000/api/users/filter

15. Block User (PATCH): http://localhost:5000/api/users/block-user/(User ID)

16. Unblock User (PATCH): http://localhost:5000/api/users/unblock-user/(User ID)

17. Verify Email (POST): http://localhost:5000/api/users/verify-email

18. Resend Verification (POST): http://localhost:5000/api/users/resend-verification

19. Upload Profile Picture (POST): http://localhost:5000/api/users/upload-profile-picture
                    (Note: key=picture, type=File)

20. Delete Account (DELETE): http://localhost:5000/api/users/delete-account
